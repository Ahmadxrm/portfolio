# Professional Portfolio - Ahmad Sulaiman

A modern, responsive, and minimalist portfolio website designed for a Software Developer.

## Overview
- **Developer**: Ahmad Sulaiman
- **Tech Stack**: HTML5, CSS3, JavaScript (Vanilla)
- **Design Theme**: Dark Navy + Teal Accent (Professional, Developer-focused)

## Features
- **Responsive Design**: Works seamlessly on Mobile, Tablet, and Desktop.
- **Sections**: Home/Hero, About Me, Skills, Projects, Contact.
- **Clean UI**: Custom CSS variables, smooth scrolling, and subtle hover effects.

## Project Structure
```
/portfolio
├── index.html   # Main structure
├── style.css    # Styling and responsiveness
└── script.js    # Interactivity (Mobile menu, dynamic year)
```

## How to Run Locally
1. Clone the repository or download the folder.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
   - Tip: For live reloading during development, use the "Live Server" extension in VS Code.

## Deployment Instructions

### Option 1: GitHub Pages (Recommended for Static Sites)
1. Push this code to a GitHub repository (e.g., `username.github.io` or `portfolio`).
2. Go to **Settings** > **Pages**.
3. Select `main` branch as the source.
4. Your site will be live at `https://[username].github.io/[repo-name]`.

### Option 2: Netlify / Vercel
1. Create an account on [Netlify](https://www.netlify.com/) or [Vercel](https://vercel.com/).
2. Drag and drop the `portfolio` folder onto their dashboard.
3. It will deploy instantly.
